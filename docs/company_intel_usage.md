# 企业招聘情报使用说明

1. 启动后端：在 `backend` 目录运行 `uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload`。
2. 启动前端：在 `frontend` 目录运行 `pnpm install` 后执行 `pnpm dev`。
3. 打开前端左侧菜单「企业招聘情报 / 公司查询」。
4. 输入公司名称，选择平台、城市和岗位关键词。
5. 默认使用「模拟数据」模式，点击「开始查询」后会展示岗位结果和招聘活跃度评分。
6. 需要真实平台查询时，先进入「平台账号」，点击对应平台的「打开登录」，在弹出的浏览器中完成登录或验证码处理。
7. 登录后点击「检测」，状态显示「可能已登录」后，在公司查询页选择「真实优先」或「仅真实」。
8. 在「查询任务」中可查看历史任务，在「招聘报告」中可导出 Excel。

## 查询模式

- `模拟数据`：只使用本地模拟适配器，适合演示。
- `真实优先`：优先使用 Playwright 真实平台查询；遇到未登录、验证码或平台阻断时保留提示并降级为模拟结果。
- `仅真实`：只使用真实平台查询；未登录或验证码时不会生成模拟岗位。
